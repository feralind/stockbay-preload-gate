# Local logo cache (optional)

Drop PNG files here for instant loading without hitting Google favicons.

## Stocks
`AAPL.png`, `F.png`, `HMC.png`, `ADDYY.png`, etc. — filename = ticker symbol.

## Banks / domains
`domains/chase_com.png`, `domains/bankofamerica_com.png` — dots replaced with underscores.

If a local file is missing, the app falls back to Google → DuckDuckGo favicons automatically.
